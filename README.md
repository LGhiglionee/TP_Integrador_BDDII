# TP_Integrador_BDDII


Descargar PIP INSTALL streamlit
Para correr pagina web: streamlit run app_web.py
Descargar pip install redis
Descargar pip install pymongo
Descargar pip install cassandra-driver
Descargar pip install neo4j

Para conectar con Neo4j:
Se debe ir a Neo4j desktop, crear una instancia con usuario por defecto y contrasena cursusNeo4j, y tambien instalar neo4j en python (pip install neo4j)

Para conectar con Cassandra y redis:
Se deben iniciar las instancias en docker con el nombre por defecto dado por docker

Para correr el programa:
streamlit run app_web.py en la terminal
O
python -m streamlit run app_web.py
